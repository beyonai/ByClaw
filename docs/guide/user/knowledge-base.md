# 知识库使用指南

知识库是 ByClaw 的核心功能之一，用于管理企业知识文档，实现智能问答。

## 什么是知识库？

知识库是一个文档集合，可以：
- 存储和管理企业文档
- 被 AI 智能检索和引用
- 支持多种文档格式
- 实现权限控制

## 创建知识库

1. 点击 **"知识库"** 导航
2. 点击 **"新建知识库"**
3. 填写信息：
   - **名称** - 知识库名称
   - **描述** - 知识库用途说明
   - **可见性** - 公开/私有
4. 点击 **"创建"**

## 上传文档

### 支持的格式

| 格式 | 扩展名 | 说明 |
|------|--------|------|
| PDF | .pdf | 便携式文档 |
| Word | .doc, .docx | Microsoft Word |
| 文本 | .txt | 纯文本 |
| Markdown | .md | Markdown 文档 |
| Excel | .xls, .xlsx | 表格数据 |
| PPT | .ppt, .pptx | 演示文稿 |

### 上传步骤

1. 进入知识库详情页
2. 点击 **"上传文档"**
3. 选择文件或拖拽上传
4. 等待处理完成

**处理流程：**
- 上传中 → 解析中 → 向量化 → 已完成

### 批量上传

支持同时上传多个文件：
- 拖拽多个文件到上传区域
- 或按住 Ctrl 选择多个文件

## 文档管理

### 查看文档

- 列表视图：显示所有文档
- 点击文档查看详情和分段内容

### 文档操作

| 操作 | 说明 |
|------|------|
| 刷新 | 重新处理文档 |
| 删除 | 移除文档 |
| 下载 | 下载原始文件 |
| 预览 | 在线预览内容 |

### 文档状态

| 状态 | 说明 |
|------|------|
| 🟡 处理中 | 正在解析和向量化 |
| 🟢 已完成 | 可以正常检索 |
| 🔴 失败 | 处理出错，需重新上传 |

## 知识库设置

### 基本信息

- **名称** - 修改知识库名称
- **描述** - 更新描述信息
- **封面** - 设置知识库封面

### 权限管理

- **公开** - 所有用户可访问
- **私有** - 仅创建者和授权用户可访问
- **指定用户** - 添加特定用户权限

### 检索设置

- **相似度阈值** - 控制检索严格程度
- **返回数量** - 每次检索返回的文档数
- **重排序** - 启用重排序优化结果

## 使用知识库

### 在对话中关联

1. 进入 **"对话"** 页面
2. 点击输入框上方的 **"知识库"** 按钮
3. 选择一个或多个知识库
4. 输入问题，AI 会基于知识库回答

### 检索效果优化

**提高检索准确度：**
- 使用清晰的文档命名
- 确保文档内容质量
- 适当调整相似度阈值
- 定期更新知识库内容

## 最佳实践

### 文档组织

- 按主题/部门组织多个知识库
- 避免单个知识库文档过多（建议 < 1000 个）
- 定期清理过期文档

### 内容质量

- 优先上传结构化文档
- 避免扫描版 PDF（图片文字无法提取）
- 确保文档编码正确

### 权限管理

- 敏感文档使用私有知识库
- 定期检查权限设置
- 离职员工及时移除权限

## 故障排查

**文档解析失败？**
- 检查文件是否损坏
- 确认不是扫描版 PDF
- 尝试转换为其他格式重新上传

**检索结果不准确？**
- 降低相似度阈值
- 检查知识库内容相关性
- 尝试不同的问题表述

**处理速度慢？**
- 大文档需要更长时间
- 检查系统资源使用情况
- 考虑分批上传
