# AI 对话功能

ByClaw 提供强大的 AI 对话能力，支持多轮对话、文件分析、知识库关联等功能。

## 基础对话

### 开始对话

1. 点击左侧 **"新建对话"**
2. 输入框中输入问题
3. 按 Enter 或点击发送

### 对话技巧

| 技巧 | 示例 |
|------|------|
| 清晰表达 | "请解释什么是 RAG" |
| 提供上下文 | "基于上述内容，总结要点" |
| 分步提问 | 复杂问题拆分为多个小问题 |
| 指定格式 | "请用表格对比 A 和 B" |

## 高级功能

### 文件上传

支持上传文件让 AI 分析：

1. 点击输入框旁的 **📎 附件按钮**
2. 选择文件（支持 PDF、Word、图片）
3. 输入你的问题
4. 发送

**使用场景：**
- 总结长文档
- 提取表格数据
- 分析图片内容
- 代码审查

### 知识库关联

让 AI 基于企业知识库回答：

1. 点击输入框上方的 **"知识库"** 按钮
2. 选择要关联的知识库
3. 输入问题
4. AI 会优先从知识库中检索答案

### 提示词模板

快速使用预设提示词：

1. 在输入框输入 `/`
2. 选择提示词模板：
   - `/翻译` - 翻译文本
   - `/总结` - 总结内容
   - `/代码` - 生成代码
   - `/解释` - 解释概念

### 模型切换

如系统配置了多个 AI 模型：

1. 点击对话设置
2. 选择 AI 模型
3. 新消息将使用选中的模型

## 对话管理

### 历史对话

- 左侧列表显示历史对话
- 点击可继续之前的对话
- 长按/右键可删除对话

### 对话设置

点击对话顶部的 **⚙️ 设置**：

- **重命名** - 修改对话标题
- **清空** - 清空当前对话内容
- **导出** - 导出对话记录（Markdown/JSON）

### 收藏消息

1. 将鼠标悬停在消息上
2. 点击 **⭐ 收藏按钮**
3. 在 "收藏" 页面查看所有收藏

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Enter` | 发送消息 |
| `Shift + Enter` | 换行 |
| `/` | 打开提示词模板 |
| `Ctrl + K` | 新建对话 |
| `Esc` | 取消生成 |

## 常见问题

**Q: AI 回答不准确怎么办？**

A: 
- 检查是否关联了正确的知识库
- 尝试更清晰地描述问题
- 提供更多上下文信息

**Q: 文件上传失败？**

A:
- 检查文件大小（通常限制 50MB）
- 确认文件格式是否支持
- 检查网络连接

**Q: 对话响应慢？**

A:
- 检查网络连接
- 复杂问题可能需要更多处理时间
- 联系管理员检查系统负载
