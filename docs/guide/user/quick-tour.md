# 快速上手指南

5 分钟了解 ByClaw 核心功能。

## 首次登录

1. 打开 ByClaw 地址（如 http://localhost:8080）
2. 使用默认管理员账号登录：
   - 用户名：`admin`
   - 密码：（部署时设置的默认密码）
3. 首次登录后建议修改密码

## 界面概览

```
┌─────────────────────────────────────────────────────────────┐
│  Logo    首页   对话   知识库   数字员工   工作中心    [用户] │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│                    [主内容区域]                              │

│                                                             │
└─────────────────────────────────────────────────────────────┘
```

主要导航：
- **首页** - 仪表盘和快捷入口
- **对话** - AI 智能对话
- **知识库** - 企业知识管理
- **数字员工** - AI Agent 管理
- **工作中心** - 任务和协作

## 核心功能体验

### 1. AI 对话

点击顶部导航的 **"对话"**，开始与 AI 助手聊天：

1. 在输入框输入问题
2. 按 Enter 或点击发送按钮
3. AI 会实时流式回复

**小技巧：**
- 支持多轮对话，AI 会记住上下文
- 可以上传文件让 AI 分析
- 使用 `/` 快速选择提示词模板

### 2. 知识库

点击 **"知识库"** 管理企业知识：

1. **创建知识库** - 点击 "新建知识库"
2. **上传文档** - 支持 PDF、Word、TXT、Markdown
3. **智能问答** - 在对话中选择知识库，AI 会基于知识库回答

**支持的文档格式：**
- PDF (.pdf)
- Word (.doc, .docx)
- 文本 (.txt)
- Markdown (.md)

### 3. 数字员工

点击 **"数字员工"** 配置 AI Agent：

1. **创建员工** - 定义角色和能力
2. **配置工具** - 添加可使用的工具
3. **分配任务** - 让数字员工自动执行任务

### 4. 个人设置

点击右上角头像：
- **个人资料** - 修改昵称、头像
- **密码** - 修改登录密码
- **退出** - 安全退出
