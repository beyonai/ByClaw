# 资源权限说明

本文说明资源中心五类资源在个人 tab、企业 tab 下的展示范围与操作权限口径。

## 资源类型

五类资源包括：

- 数字员工/个人助理：`DIG_EMPLOYEE`
- 知识：`KG_DOC`、`KG_DB`、`KG_QA`、`KG_TERM`
- 工具：`AGENT`、`MCP`、`TOOLKIT`、`TOOL`、`MCP_TOOL`
- 对象：`OBJECT`
- 视图：`VIEW`

## 列表范围

个人 tab：

- 个人助理查询 `owner_type=personal` 中我创建、我管理、我使用的资源，并额外展示当前默认个人助理和默认超级助手。
- 个人知识查询个人资源，并额外展示当前用户绑定的默认个人知识库。
- 个人工具、对象、视图查询个人资源中我创建或我被授权的资源。
- 命中当前用户使用黑名单的资源不展示。

企业 tab：

- 企业数字员工查询 `owner_type=enterprise` 的数字员工，不要求必须由当前用户创建或授权给当前用户。
- 企业知识、工具、对象、视图按企业资源全量查询。
- 命中当前用户使用黑名单的资源不展示。
- 组织筛选时按资源归属组织 `man_org_id` 收口。

## 补充说明

- 有管理权限指满足任一条件：平台管理员、资源创建人、资源归属组织管理员、对该资源有有效 `ALLOW_MANAGE` 授权。
- 使用授权维护权限复用管理权限口径；当前用户绑定的默认个人资源也允许维护使用授权。
- 使用申请的通用条件包括：资源允许门户授权、当前用户不是创建人、当前用户未命中使用黑名单、当前用户未已有使用权限、当前用户未已有待审核申请。
- 个人 tab 下知识、工具、对象、视图不走使用申请/使用审核链路，只允许有管理权限的人主动授权别人使用。
- 个人助理、默认个人助理、默认超级助手不允许管理授权、使用申请和使用审核。
- 默认超级助手是登录自动初始化的个人底座资源，`resource_code` 固定为 `{userCode}_main`，不允许授权给别人使用。
- 当 `DATASET_SYSTEM=WHALE_AGENT` 时，知识和工具均由外部智能体体系通过后端接口发布，本系统不允许编辑基础信息或注销数据。
