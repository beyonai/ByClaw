# 架构决策记录 (ADR)

本文档记录 ByClaw 项目的重要架构决策。

## 什么是 ADR？

架构决策记录（Architecture Decision Record，ADR）用于记录项目中重要的技术决策，包括：

- **Context** - 决策背景
- **Decision** - 做出的决定
- **Consequences** - 决策带来的影响

## 状态说明

- **Proposed** - 已提议，待讨论
- **Accepted** - 已接受，正在实施
- **Deprecated** - 已废弃，不再使用
- **Superseded** - 已被新决策取代
